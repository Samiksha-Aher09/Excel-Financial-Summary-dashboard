# Excel-Financial-Summary-dashboard
# Project Overview :- This project is an Excel-based Financial Summary Dashboard for 2023, developed to analyze and visualize key financial metrics such as assets, liabilities, revenue, expenses, cash flow, and transaction activity.
  The dashboard uses interactive Excel features to provide a clear overview of financial performance and support data-driven financial decision-making.
# Objectives
  - Analyze overall financial performance
  - Track monthly financial trends
  - Understand account-wise debit and credit distribution
  - Evaluate category-wise financial summary
  - Analyze average transaction value by account
  - Monitor financial activity by transaction type
# Key Insights
  - Balanced distribution across Assets, Liabilities, Revenue, and Expenses
  - Monthly financial values show noticeable peaks during mid-year
  - Accounts Payable and Accounts Receivable contribute significantly to total transactions
  - Inventory and Cash maintain stable transaction averages
  - Debit and Credit values remain closely aligned across categories
# Tools & Excel Features Used
  - Microsoft Excel
  - Pivot Tables & Pivot Charts
  - Slicers (Date, Category, Payment Mode, Transaction Type)
  - Advanced Excel formulas (SUMIFS, AVERAGEIFS)
  - Financial data aggregation & analysis
  - Dashboard formatting & design
# Dashboard Components
  - Account-wise Financial Summary (Debit vs Credit)
  - Activity of Accounts (Donut Chart)
  - Monthly Financial Summary (Trend Analysis)
  - Average Transaction Value by Account
  - Category-wise Financial Summary (Asset, Expense, Liability, Revenue)
# Dataset Information
  - Financial transactions dataset (2023)
  - Debit and credit records
  - Account and category classification
  - Payment modes and transaction types
  - Date-wise transaction data
# Business Use Case
  - Monitor organizational financial health
  - Identify spending and revenue trends
  - Support accounting and budgeting decisions
  - Improve financial transparency and reporting
# Future Enhancements
